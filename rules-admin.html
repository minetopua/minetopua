Головна Правила Донат Адміністрація Контакти

Правила для адміністрації

1. Загальні правила

  • 1.1 Адміністратори зобов’язані дотримуватися всіх правил, зазначених у цьому документі.
  • 1.2 Почавши виконувати обов'язки адміністратора, ви автоматично підтверджуєте свою згоду з цим набором правил.
  • 1.3 Адміністратор має право карати гравців тільки у відповідності до правил, а винятки потребують обґрунтування.
  • 1.4 Адміністрація несе відповідальність за свої дії на сервері, включно з рішеннями щодо покарань.
  • 1.5 Усі дії адміністрації мають бути прозорими та фіксуватися у логах для подальшого аналізу.
  • 1.6 Заборонено перевищувати свої повноваження, включно з необґрунтованими діями, що впливають на гравців.
  • 1.7 Адміністрація не має права поширювати інформацію про сторонні сервери або використовувати сервер для власної вигоди.
  • 1.8 Заборонено ображати, провокувати або дискримінувати гравців.

2. Обов’язки адміністратора

  • 2.1 Адміністратор повинен зберігати нейтральність у конфліктах між гравцями.
  • 2.2 Використання прав адміністратора для особистої вигоди (телепортація, видача ресурсів тощо) суворо заборонено.
  • 2.3 Заборонено змінювати або руйнувати будівлі гравців без їхнього дозволу, якщо це не пов'язано з порушенням правил.
  • 2.4 Адміністратор не має права видавати предмети, ресурси чи привілеї без дозволу старшої адміністрації.
  • 2.5 У разі виникнення проблем на сервері адміністратор зобов’язаний оперативно повідомити старшу адміністрацію.

3. Покарання

  • 3.1 Адміністратор має чітко слідувати системі покарань, прописаній у правилах.
  • 3.2 Усі покарання повинні бути обґрунтованими та супроводжуватися доказами (скриншоти, відео тощо).
  • 3.3 Заборонено використовувати свої права для необґрунтованих банів, мутацій або кікання гравців.
  • 3.4 У разі спірних ситуацій адміністратор зобов’язаний звернутися до старшої адміністрації для прийняття рішення.

4. Спілкування з гравцями

  • 4.1 Адміністратор повинен бути ввічливим та дотримуватися етики під час спілкування.
  • 4.2 Заборонено ображати або провокувати гравців у чаті чи особистих повідомленнях.
  • 4.3 Адміністратор не має права дискримінувати гравців за будь-якими ознаками.
  • 4.4 У разі конфліктів адміністрація повинна залишатися нейтральною стороною та сприяти їхньому вирішенню.

5. Використання привілеїв

  • 5.1 Заборонено використовувати адміністративні можливості для власної гри (режим бога, креатив тощо).
  • 5.2 Усі дії адміністратора повинні мати обґрунтовану мету, пов’язану з підтримкою порядку на сервері.
  • 5.3 Заборонено втручатися у процес гри гравців без необхідності.

6. Відповідальність

  • 6.1 У разі порушення правил адміністрація несе відповідальність відповідно до рішення старшої адміністрації.
  • 6.2 Повторні порушення можуть призвести до зняття адміністративних прав.
  • 6.3 Адміністратор повинен надавати звіт про свої дії за запитом старшої адміністрації.

7. Заборони

  • 7.1 Заборонено використовувати сторонні програми, скрипти або моди для отримання переваг.
  • 7.2 Заборонено розголошувати конфіденційну інформацію про сервер або гравців.
  • 7.3 Заборонено вдавати із себе старшу адміністрацію або інших членів команди.
  • 7.4 Використання адміністративних прав для спричинення шкоди серверу суворо карається.

Ці правила розроблені для забезпечення справедливості та належної роботи адміністрації. Порушення будь-якого пункту може призвести до дисциплінарних заходів.
